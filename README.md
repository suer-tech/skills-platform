# Skills Platform

Платформа для запуска универсальных AI-скиллов на любом хосте или устройстве.
Скиллы живут в репозитории, вызываются по REST API — без копирования контекста и без привязки к среде выполнения.

## Принцип

Каждый скилл — изолированный агент со своей инструкцией (SKILL.md), схемами (references/) и шаблонами (assets/).
API не нагружает контекст — скиллы загружаются и выполняются только по запросу.

- Единый набор скиллов для всех окружений
- Вызов откуда угодно: CLI, VS Code, CI/CD, web
- Контекст не перегружается — скиллы подтягиваются по необходимости
- Не требует установки агента на целевой машине

## Быстрый старт

```bash
pip install -r requirements.txt
uvicorn api.main:app --reload
curl http://localhost:8000/api/skills
```

## Скиллы

| ID | Назначение |
|---|---|
| `documentation.new-project` | Создание документации нового проекта |
| `documentation.update-docs` | Обновление документации по коду |
| `engineering.bug-hunt` | Поиск багов в проекте |

`GET /api/skills` — полный список. `GET /api/skills/{id}` — детали.
